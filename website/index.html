Concept & Mission

Determining if AI can reliably "read" a researcher's working directory and draft high-quality metadata records with minimal intervention.

Status: Early Alpha / Technology PoC.
Metadata: Strictly follows RODBUK schema.
Scope: Automated extraction & refinement loop.
Platform: Native builds for Win/Mac/Linux.

Read-Only Safety

The tool never modifies original research files. Your data remains untouched and 100% secure.

AI Reasoning

Deep analysis of LaTeX, VASP, DICOM and HDF5 to identify authors, abstracts, and methodologies.

Refinement Loop

Refine metadata through a simple conversation. Tell the tool what's missing, and it learns from context.

Installation Options

Version 0.22.22 | Released: February 23, 2026

Install from PyPI

Recommended for developers and power users

Fastest (uv)

Ultra-fast Python package manager in Rust

# Install uv first
curl -LsSf https://astral.sh/uv/install.sh | sh
uvx opendata-tool

pipx

Isolated Python app installation

# Install pipx first
python -m pip install --user pipx
pipx install opendata-tool

pip

Traditional Python installation

# Create venv first
python -m venv venv && source venv/bin/activate
pip install opendata-tool

Development Installation

Clone and install from source
$ git clone https://github.com/jochym/opendata.git
$ cd opendata
$ pip install -e ".[dev]"
$ python src/opendata/main.py
